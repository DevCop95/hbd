# 🎂 Happy Birthday · Página de Cumpleaños

Tarjeta de cumpleaños interactiva construida en HTML puro, desplegada como sitio estático con GitHub Pages.

## ✨ Características

- Pantalla de carga animada
- Mensaje personalizado con tipografía retro (Press Start 2P)
- Diseño responsivo optimizado para móvil (PWA-ready)
- Paleta de colores: púrpura, oro, rosa y cyan
- Sin dependencias externas · 100% HTML/CSS/JS
